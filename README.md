# se-nrtesting

## Describe

	se-nrtesting is a non-regression testing lib using Selenium and Allure

## Usages

	Init your npm project and install se-nrtesting

	You need 3 types of files :

		1. the 'main' file named main.test.js : ./test/main.test.js

		2. the 'lib' file where all functions as hardcoded : ./test/lib.test.js

		3. the 'scenario' files for each scenario do you want execute : like SC01, SC02, SC03...
